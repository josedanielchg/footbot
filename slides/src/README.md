# Logo de la universidad (diapositivas)

Coloca aquí el logo de la universidad (UNET) para la portada y el cierre.

- **Nombre esperado:** `logo-unet.png` (PNG con fondo transparente, recomendado).
- La portada lo incluye **automáticamente** si existe (`\IfFileExists{src/logo-unet.png}`).
- Si no está, se muestra un recuadro placeholder y la presentación **compila igual**.
